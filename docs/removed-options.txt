Removed options (sidekick-next)
Date: 2026-01-22

These settings were removed because they existed in the Settings UI but had no functional implementation (no references outside settings UI/registry), so changing them could not affect runtime behavior.

- AuraSelection
- AutoStandFD
- BuffAllowInCombat
- BuffCoordinateActors
- BuffFellowshipEnabled
- BuffGroupEnabled
- BuffRaidEnabled
- BuffRebuffWindow
- BuffSelfOnly
- CooldownSweepEnabled
- DoCombatRez
- DoOutOfCombatRez
- EmergencyHealPct
- LowResourceThreshold
- MAScanZRange
